Tischtennis Turnier – PRO+

Features:
- Minimalistischer Pro‑Hintergrund (tt_modern_bg.jpg)
- Anmeldung mit Logo‑Einflug, Countdown
- Musik startet nach erstem Klick (leise, loop, ohne Player)
- Ranking per Button in neuem Tab (ranking.html), live aktualisiert
- Admin & CSV‑Download mit Zugangsschutz (Basic Auth)

Admin-Login (Standard, änderbar per ENV):
- ADMIN_USER=admin
- ADMIN_PASS=tsv1905

Lokal:
1) npm install
2) npm start
3) http://localhost:3000

Render:
- Build: npm install
- Start: npm start
- Env: ADMIN_USER, ADMIN_PASS (optional anpassen)
