Tischtennis Turnier - PRO Paket

Enthält:
- Pro-Design Hintergrund (tt_pro_bg.jpg)
- Modernes Frontend mit Live-Ranking (höchste TTR zuerst)
- Backend (Node/Express) speichert Anmeldungen in CSV
- Admin-Page & CSV-Download

Local:
1) node & npm installiert
2) npm install
3) npm start
4) open http://localhost:3000

Deployment:
- Upload all files to GitHub
- Deploy to Render: New -> Web Service -> Connect Repo
- Build: npm install, Start: npm start
- Upload your logo.png and music.mp3 into the repo or Render files
